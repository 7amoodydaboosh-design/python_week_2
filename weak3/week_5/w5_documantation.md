1.1 Problem Statement

Develop a Python program for a café that automatically calculates a customer’s bill based on the quantity of coffee, tea, and sandwiches purchased. The program should also display a receipt showing the customer’s order and total payment.

⸻

1.2 Inputs

* Customer name
* Coffee quantity
* Tea quantity
* Sandwich quantity

⸻

1.3 Outputs

* Customer name
* Quantity of each item
* Total bill (RM)
* Receipt

⸻

1.4 Typical Process Flow

1. Start program.
2. Ask for customer name.
3. Ask for coffee quantity.
4. Ask for tea quantity.
5. Ask for sandwich quantity.
6. Calculate total price.
7. Display receipt.
8. End program.

⸻

1.5 Constraints

* Coffee price = RM8.50
* Tea price = RM6.00
* Sandwich price = RM12.00
* Quantities must be zero or positive integers.

⸻

2. Decompose the Problem

The program can be divided into these smaller tasks:

* Get customer information.
* Get quantities for each item.
* Calculate the total bill.
* Print the receipt.
* End the program.
